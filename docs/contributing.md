PRs welcome. The full guide lives at the repo root so it shows up on GitHub's contribute prompts:

> **[`CONTRIBUTING.md`](https://github.com/Vijay431/Orkestra/blob/main/CONTRIBUTING.md){: .btn .btn-primary }**
